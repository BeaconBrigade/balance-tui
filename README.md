# `balance-tui`

Balance tui is a simple cli program to balance chemical equations.

Usage by argument passsing:

```shell
$ cmbl "Fe + O2 -> Fe2O3"
4Fe + 3O2 -> 2Fe2O3
```

Or... invoke without arguments to start the tui
